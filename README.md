## Getting Started

```bash
git clone https://github.com/hmhard/wolaita-bible-website.git
cd wolaita-bible-website
npm install
# or 
pnpm install
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
